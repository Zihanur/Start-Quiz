Map() Method: Map method call a function in the Array and check every array element at last return every element. If need return in the Array then use Map.

ForEach() Method: ForEach method call a function in the Array and check every array element but on return every element. If check every element in the Array but no return then use ForEach.

Filter() Method: Filter method return all elements of the array that pass the condition in the call back function.

Find() Method: Find method return the 1st element of the array that match the condition.

Local Store: Local Store is property, that alow JavaScript sites to save value and this value-key in the browser storage. The stored data is not deleted when the browser closed.

Session Store: Session Store is property, it same of Local Store that alow JavaScript sites to save value and this value-key in the browser storage. But when the browswe the data will be deleted. It just a session when browser will run.

Cookie: Our computer save our sort data. Example: our name, home address, mobile number, email etc. It is cookie. When we back the website cookie remember this name, mail or phone number. Cookie expire at certain time.

JavaScript Event Loop: When JavaScript call then JS Machine waiting for execution in the queue. Event loop job is to monitor the call stack the queue. Event Loop is a process that wait for the call stack to be clear before pushing callback from the task queue. This is a Loop process.
